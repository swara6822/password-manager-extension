# Tiny Password Manager (Chrome Extension)

An educational Chrome extension built to learn browser extensions,
storage, and basic security concepts.

## Features
- Save website credentials locally
- Master password protection (hashed, not stored in plain text)
- Manual autofill for login forms
- Password generator

## Disclaimer
This project is for learning purposes only.
Do not use it to store real or sensitive passwords.

## Status
Work in progress

## Setup
1. Clone the repository
2. Open chrome://extensions
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the project folder
